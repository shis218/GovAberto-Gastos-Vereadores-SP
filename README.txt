Codigos gerados para a disciplina de Governo Aberto - ACH3778 da faculdade EACH.

>Pasta Codigo Python
Na pasta do codigo em python possui um arquivo único(Python GovAberto.ipynb) usado para executar a leitura e tratamento da pagina web da camara municipal de São Paulo.Utilizamos o JupyterNotebook do pacote Anaconda3 para produção e execução desse artefato.
Dentro dessa pasta ainda possui os resultados gerados tanto em CSV quanto em JSON. E em caso de queda do site oficial, tambem deixamos uma copia de uma das paginas para consulta.


>Pasta Codigo API
Na pasta API master, existe o código server.js para executalo é necessário instalar o node js além dos pacotes nodemon e o express. os dados para a conexão no banco estão usuario e senha estão disponíveis no código. A API está configurada para execução localhost.
